# FileCompare
# (C# 코딩) 파일 비교 프로그램
## 개요
-C# 프로그래밍학습
-1줄소개: 사용자키보드입력을받아서처리하는프로그램
-사용한플랫폼: 
	-C#, .NET Windows Forms, Visual Studio, GitHub
	-사용한컨트롤:-Label, TextBox, ListBox, Button
	-사용한기술과구현한기능:
		-Visual Studio를이용하여UI 디자인
		-string 클래스를이용한사용자입력데이터처리
		-DateTime클래스를이용한현재시간정보구하기

![실행화면](img/screenshot-1.png)

-구현한내용(위그림참조)
	-UI 구성: Label(앱이름표시), TextBox2개(아이디, 패스워드)
